# A반 Ex1 - equals/hashCode 응용

`Student(no, name)` 클래스를 만들고, 학번(`no`)이 같으면 같은 학생으로 판단되도록 `equals()`와 `hashCode()`를 재정의하세요.

## 요구사항
1. `HashSet<Student>`에 아래 데이터를 저장하세요.
2. 최종 저장된 학생 수를 출력하세요.
3. 학번 오름차순으로 학생 목록을 출력하세요.

## 데이터
- (1001, "Kim")
- (1001, "Kim-dup")
- (1002, "Lee")
- (1003, "Park")
- (1002, "Lee-dup")
