# 2026년도 3번 - 게시판 글 관리 프로그램

다음은 게시글 목록 예시이다.

```java
List<Post> posts = Arrays.asList(
    new Post(1, "공지사항입니다", "관리자", 150, PostType.NOTICE),
    new Post(2, "자바 질문", "홍길동", 80, PostType.QNA),
    new Post(3, "자유게시판 글", "김철수", 120, PostType.FREE),
    new Post(4, "스프링 질문", "홍길동", 200, PostType.QNA),
    new Post(5, "점심 추천", "이영희", 50, PostType.FREE)
);
```

위 데이터를 이용하여 다음 기능을 구현하시오.

1. 조회수 100 이상인 게시글만 출력하시오.
2. 게시판 유형별 글 수를 출력하시오.
3. 조회수 기준 내림차순 정렬 결과를 출력하시오.
4. 작성자별 게시글 수를 출력하시오.
5. 제목에 "질문"이 포함된 게시글만 출력하시오.
6. 가장 조회수가 높은 게시글을 출력하시오.

조건:

- `PostType`은 `enum`으로 작성한다.
- 게시글의 중복 여부는 글 번호 기준으로 판단한다.

설명과 함께 출력 결과를 보여주는 실행 파일을 작성하시오.
