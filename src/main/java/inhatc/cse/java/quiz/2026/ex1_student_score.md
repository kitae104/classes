# 2026년도 1번 - 학생 성적 처리 프로그램

다음은 학생 정보가 저장된 예시이다.

```java
List<Student> students = Arrays.asList(
    new Student(1001, "김철수", 90, Grade.A),
    new Student(1002, "이영희", 75, Grade.C),
    new Student(1003, "박민수", 85, Grade.B),
    new Student(1004, "최지우", 95, Grade.A),
    new Student(1002, "이영희", 75, Grade.C)
);
```

위 데이터를 이용하여 다음 기능을 구현하시오.

1. 점수가 80점 이상인 학생만 출력하시오.
2. 이름 기준 오름차순 정렬 결과를 출력하시오.
3. 점수 기준 내림차순 정렬 결과를 출력하시오.
4. 학점별 학생 수를 구하여 출력하시오.
5. 중복 학생을 제거한 결과를 출력하시오.
6. 점수가 가장 높은 학생 1명을 출력하시오.

조건:

- `Grade`는 `enum`으로 작성한다.
- 학생의 중복 여부는 학번(학생 번호) 기준으로 판단한다.
- 점수 정렬은 `Comparator`를 활용한다.

설명과 함께 출력 결과를 보여주는 실행 파일을 작성하시오.
